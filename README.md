# KID_LOCK-A-Privacy-Aware-Parental-control-system
KID_LOCK: A Privacy Aware &amp; Parental Control System is an Android app that creates a safe digital space for children. Parents can monitor usage, block or allow apps, and set limits, while kids access only approved apps and learn privacy through interactive lessons. All rules are enforced securely using a local control engine.
